<h1>Ulam's Square</h1>

### [Release Build](https://github.com/ItsCam32/UlamsSquare/releases/tag/v1.0.0)

<h2>Description</h2>
Ulam's Spiral is a visualisation of the interesting behaviour of prime numbers when highlighted against non-primes in a spiraling pattern to form a square; the lines that are created correspond with quadratic polynomials.
<br>
<br>
<b>Simulated in the Unity engine.</b>
